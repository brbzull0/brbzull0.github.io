SOCKS¶

proxy.process.socks.connections_currently_open¶
Collection:global
Type:gauge
Datatype:integer
Gathered:No

Provides the number of SOCKS proxy connections currently opened with the running instance of Traffic Server.

proxy.process.socks.connections_successful¶
Collection:global
Type:counter
Datatype:integer
Gathered:No

Indicates the number of SOCKS connections to Traffic Server which have succeeded since statistics collection began.

proxy.process.socks.connections_unsuccessful¶
Collection:global
Type:counter
Datatype:integer
Gathered:No

Indicates the number of attempted SOCKS connections to Traffic Server which have failed since statistics collection began.

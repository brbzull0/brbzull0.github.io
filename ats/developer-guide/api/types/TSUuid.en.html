TSUuid¶

Synopsis¶

#include <ts/apidefs.h>
type TSUuid¶

An opaque pointer to an internal representation of a UUID object.

Description¶

The UUID object is managed and created via the various UUID APIs. You can not access nor modify this object using anything but the provided APIs.

See Also¶

TSUuidCreate(3ts), TSUuidInitialize(3ts), TSUuidDestroy(3ts), TSUuidCopy(3ts), TSUuidStringGet(3ts), TSUuidVersionGet(3ts), TSUuidStringParse(3ts), TSProcessUuidGet(3ts)

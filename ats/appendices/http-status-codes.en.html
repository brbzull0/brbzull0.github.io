HTTP Status Codes¶

The following standard HTTP response messages are provided for your information.

Code Description
200 OK
202 Accepted
204 No Content
206 Partial Content
300 Multiple Choices
301 Moved Permanently
302 Found
303 See Other
304 Not Modified
400 Bad Request
401 Unauthorized; retry
403 Forbidden
404 Not Found
405 Method Not Allowed
406 Not Acceptable
408 Request Timeout
500 Internal Server Error
501 Not Implemented
502 Bad Gateway
504 Gateway Timeout

More detail can be found in RFC2616 which defines the full range of standardized HTTP/1.1 response codes.
